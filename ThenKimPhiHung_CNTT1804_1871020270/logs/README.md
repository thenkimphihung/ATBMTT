# Logs minh chứng

Thư mục này dùng để lưu log chạy thật của **nhóm 2 người**.

Gợi ý đặt tên file:
- `01-happy-path-member1.txt`
- `02-happy-path-member2.txt`
- `03-tamper.txt`
- `04-wrong-key.txt`
- `05-header-error.txt`

Mỗi file log nên cho thấy:
- thời điểm chạy,
- ai trong nhóm thực hiện ca demo,
- input / tình huống,
- kết quả nhận được,
- nếu lỗi thì lỗi xuất hiện ở đâu.
